# LunchBreakCalculator
LunchBreakCalculator app based on javascripts react framework.

## dev

```shell
#init
npm init
#install packages for dev
npm install react react-dom

#coding
touch LunchBreakCalculator.js
touch index.js
touch index.html

#start this web app
npm start
```
